## Installation
Clone this project from GitHub to your local machine.

Install mysql2.

To install dependencies, run the following in your Command Line Interface (e.g., Terminal or Bash):
- npm init
- npm install bcrypt
- npm install dotenv
- npm install sequelize

## Usage
Then run the following command:
- node index
